# llmTranslyator

## О проекте

**llmTranslyator** — персональное приложение для самостоятельного обучения английскому языку с помощью локальной языковой модели.

> **Это публичная шаблонная версия.** Ключевые части реализации (промпты, алгоритм адаптации прогресса, оркестрация WebSocket) заменены на заглушки. Готовая реализация развёрнута на домашнем сервере для личного использования.

---

## Как работает приложение

1. Пользователь вводит тему на любом языке и выбирает уровень CEFR (A1–C1)
2. Локальная LLM генерирует короткий текст на английском с русским переводом
3. Пользователь включает микрофон и пересказывает текст вслух
4. Аудио транскрибируется локально через Whisper
5. LLM анализирует пересказ и возвращает структурированную обратную связь:
   - грамматические ошибки с объяснением на русском
   - предложения по словарю
   - пропущенные ключевые моменты
   - общий комментарий и оценка 0–100
6. Система запоминает повторяющиеся трудности и учитывает их при генерации следующих текстов

## Инфраструктура

Проект полностью бесплатен в эксплуатации и работает на собственном железе:

```
[Браузер] → http://HOME_SERVER_IP:3000
                    │
              [Домашний сервер] (CPU, без GPU)
              ├── screen "frontend"  → Nuxt.js SSR (Node.js)
              └── screen "backend"   → FastAPI + uvicorn
                           │
                           └──→ http://WORKSTATION_IP:11434
                                    │
                               [Рабочий ПК]
                               NVIDIA RTX 5060 (16 GB VRAM)
                               └── Ollama → qwen2.5:7b
```

- **LLM** запущена локально на ПК с RTX 5060 через [Ollama](https://ollama.com)
- **STT (Whisper)** работает на домашнем сервере в CPU-режиме (`faster-whisper small int8`)
- **База данных** — SQLite, файл на сервере
- **Деплой** — две `screen`-сессии, автозапуск через
- Ни один запрос не уходит в интернет после установки

## Технический стек

### Backend

| Компонент     | Технология                                                        |
| ------------- | ----------------------------------------------------------------- |
| API Framework | FastAPI + uvicorn                                                 |
| Real-time     | WebSockets (FastAPI native)                                       |
| LLM           | [Ollama](https://ollama.com) + qwen2.5:7b                         |
| STT           | [faster-whisper](https://github.com/SYSTRAN/faster-whisper) small |
| База данных   | SQLite + SQLAlchemy async + aiosqlite                             |
| Миграции      | Alembic                                                           |

### Frontend

| Компонент | Технология                        |
| --------- | --------------------------------- |
| Framework | Nuxt.js 3 (SPA mode) + TypeScript |
| UI        | Vuetify 3                         |
| State     | Pinia                             |
| Real-time | WebSocket                         |
